# System Design Lab Website

A Next.js website for the System Design Laboratory.

## Technologies Used
- Next.js 14
- React
- Tailwind CSS

## Getting Started

First, install dependencies:
```bash
npm install
```

Then, run the development server:
```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the website.
